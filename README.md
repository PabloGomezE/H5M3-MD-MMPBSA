# H5M3 Mosaic Hemagglutinin - Structural and Dynamic Characterization

Molecular dynamics simulations and MM-PBSA binding energy analysis of H5M3 
mosaic vaccine antigen against HPAI H5N1 clade 2.3.4.4b.

## Contents

- `MDP_files/` - GROMACS MD parameter files (minimization, equilibration, production)
- `Rundata/` - Analysis results:
  - `trimmer_RMSD/` - Root mean square deviation
  - `trimmer_RMSF/` - Root mean square fluctuation
  - `trimmer_AASA/` - Antibody accessible surface area
  - `trimmer_HBONDS/` - Hydrogen bond analysis
  - `complex_MMPBSA/` - MM-PBSA binding free energy calculations
  - `eigenvalues/` - Principal component analysis
- `Scripts/` - Analysis Python scripts

## Systems Analyzed

- H5M3 (mosaic hemagglutinin)
- D1.1 (wild-type reference)
- Astrakhan 3212/2020 (wild-type reference)
